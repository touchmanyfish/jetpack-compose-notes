```kotlin
@Composable
fun demo() {
    Text(
        // 看这里！
        modifier = Modifier.height(40.dp).fillMaxWidth().background(Color.Blue),
        text = "item",
        fontSize = 30.sp,
        color = Color.Blue
    )
}
```

## 目标
让可组合函数的参数默认不换行，除非到了最大长度

## 设置
1. Setting-Editor-CodeStyle-Kotlin
2. 找到Compose选项卡
3. 取消勾选：Compose formatting-Enable Compose formatting for Modifiers
